The essential applications for Ubuntu

There are plenty of apps that you can install for Ubuntu. With it's increasing popularity and easier to use interface, the open software community has collaborated to bring some of your favourite apps of Ubuntu. You can click on the name of the respective application to visit the official website.

1. Chromium

While Ubuntu comes in-built with the Firefox web browser, some of you might not be comfortable in letting go of Chrome. Chromium (open source alternative of Chrome) is just that. It is fast, smooth and can synch all your bookmarks and settings just like Chrome.

2. Thunderbird

One of the most popular desktop email client is available on Ubuntu. It has some of the most powerful features like syncing multiple accounts, enabling you to read your emails offline and much more!

3. Skype

A lot of people rely on video conferencing these days and Skype seems to be the primary choice among users. Luckily, it's available on Ubuntu as well.

4. VLC Media Player

VLC media player just plays about anything. Ubuntu comes with an inbuilt player but nothing beats VLC as it gives users a lot more control.

5. Clementine

Rhythmbox is the currently the inbuilt audio player in Ubuntu and it's great! However, if you're looking for some more option, Clementine is certainly one of the best audio players out there. It's easy to use and hassle free. There's also Banshee, which itself is a powerful audio player.

6. Wine

One of the reasons why some users are hesitant to choose Ubuntu over Windows is the application compatibilty. There are still some applications that have not been ported on Ubuntu. However with Wine, you can install your favourite applications on Ubuntu and say goodbye to Windows!

7. Virtualbox

Virtualbox has been the best choice when it comes to a virtual environment. With Virualbox, you can run Windows on your Ubuntu device, in case you need to use Windows for some tasks.
